# ROSnode Running on PC Host
#### Step 1: create and build a catkin workspace:
```
mkdir -p ~/rosnode_pc-arduino_ws/src
```
```
cd ~/rosnode_pc-arduino_ws
```
```
catkin_make
```
```
catkin_make install
```
#### Step 2: Clone catkin package
```
cd ~/rosnode_pc-arduino_ws/src
```
```
git clone
```
